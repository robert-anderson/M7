11 /*
12  * <bra| ...cre... ...ann... |ket>
13  * the m_ann array contains the spin orbital indices in the ket but not the bra
14  * the m_cre array contains the spin orbital indices in the bra but not the ket
15  *
16  * connect: given two determinants (ket, bra), compute the ann and cre arrays
17  * which transforms ket into bra.
18  *
19  * excite: given a ket determinant, copy its value to a given bra determinant,
20  * then apply the internal ann and cre arrays to it.
21  *
22  * zero: reset the counters
23  *
24  * add_cre: append creation index
25  *
26  * add_ann: append annihilation index
27  *
28  * add: append annihilation/creation index pair
29  */
